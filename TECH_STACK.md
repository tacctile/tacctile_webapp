# Technology Stack - Ghost Hunter Toolbox

## Core Framework
- **Electron**: Desktop application wrapper
- **Node.js**: Backend runtime for main process
- **React 18**: Frontend UI library with hooks
- **TypeScript**: Type-safe JavaScript
- **Vite**: Build tool and dev server

## Media Processing
- **FFmpeg**: Video/audio processing and analysis
- **Web Audio API**: Real-time audio processing
- **Canvas API**: Image manipulation and rendering
- **WebGL**: GPU-accelerated graphics processing
- **MediaStream API**: Camera and microphone access

## AI/Machine Learning
- **TensorFlow.js**: Browser-based ML inference
- **ONNX Runtime Web**: Cross-platform ML runtime
- **OpenCV.js**: Computer vision processing
- **Whisper.js**: Audio transcription and analysis

## Hardware Integration
- **Node SerialPort**: USB/Serial device communication
- **Node Bluetooth**: Bluetooth device integration
- **WebUSB**: Browser-based USB device access
- **WebBluetooth**: Browser-based Bluetooth access
- **Node Native Modules**: C++ addons for performance

## Data & Storage
- **SQLite3**: Local database for evidence storage
- **better-sqlite3**: Fast synchronous SQLite bindings
- **IndexedDB**: Browser-side data storage
- **Node.js File System**: File operations and management

## Real-time Communication
- **WebRTC**: Peer-to-peer video/audio streaming
- **WebSocket**: Real-time data communication
- **Socket.io**: WebSocket abstraction layer
- **UDP/TCP**: Direct network protocols for devices

## UI/Styling
- **CSS3**: Modern styling with Grid/Flexbox
- **Tailwind CSS**: Utility-first CSS framework
- **React Router**: Client-side routing
- **Framer Motion**: Advanced animations
- **Chart.js**: Data visualization and charts

## Development Tools
- **ESLint**: Code linting and quality
- **Prettier**: Code formatting
- **Jest**: Unit testing framework
- **Electron Builder**: App packaging and distribution
- **TypeScript Compiler**: Type checking and compilation

## Streaming & Protocols
- **RTSP**: IP camera streaming protocol
- **RTMP**: Live streaming protocol
- **HLS**: HTTP Live Streaming
- **WebM/H.264**: Video codecs
- **Opus/AAC**: Audio codecs

## Performance Libraries
- **Web Workers**: Background processing
- **SharedArrayBuffer**: Shared memory between threads
- **WebAssembly**: High-performance computing
- **GPU.js**: GPU acceleration for JavaScript

## Security & Encryption
- **Node.js Crypto**: Encryption and hashing
- **bcrypt**: Password hashing
- **jsonwebtoken**: JWT token handling
- **HTTPS/TLS**: Secure communications

## Package Management
- **npm**: Package manager
- **Electron Forge**: Electron toolchain
- **Native Dependencies**: Platform-specific builds
- **Cross-platform Compatibility**: Windows/Mac/Linux support
